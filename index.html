VB Mileage Calculator

This course gave me the opportunity to work with Visual Basic in the Visual Studio 2017 IDE. This assignment required the construction of a GUI application used to dynamically calculate the reimbursement of travel expenses incurred by a company’s employees. It tracks variables including distance traveled (km), regular reimbursement rate ($/km), flat reimbursement rate ($/km), number of days employees qualified for the flat reimbursement amount, and total number of days spent traveling.

JFrame Veterinary Record Updater

In preparation for the final exam in this course, I prepared this GUI application using Java, its JFrame package, and the Eclipse IDE. The intention was to create a visual user interface for a veterinary office to sort through an array of veterinary records (objects of type custom class “Cat”) and input data for each. I am quite proud of this application as I constructed it completely on my own by integrating concepts from previous class assignments.

C Video Rental File System

The final project for this course gave me the opportunity to utilize my knowledge of C programming to construct a text-based database application for a video store. Using the application, users can input data for each movie available for rent, such as title, director, release date, etc. Each entry is appended to the end of a linked list, which in turn can be saved as a binary file. The application can then be used to load a previously saved binary file to work with all the movie entries saved to it. This was by far the most difficult program for me to create and required me to utilize many of my research skills to complete.

Figma Prototyping

This work sample is the mock-up of a mobile application, GotSpot?, my group designed for our final project. This was a project of rather large scale that required team collaboration to complete. I lead our team by assigning tasks to each member, performing regular check-ins with them, and updating our Trello board as tasks were completed. The result was a working digital prototype created using Figma. We incorporated the six principles of good user interface (UI) design to create a mobile application that allows motorists to rent parking spots from other app users.

Visio 2016 ERD Construction

This work sample demonstrates my ability to logically conceptualize a relational database by constructing an entity-relationship diagram (ERD) using Microsoft Visio. This is a critical step in database design as it provides an opportunity to identify the entities, attributes, and relationships involved in an information system before any code is written.

SWOT Analysis

My systems analysis course taught me how to perform a SWOT analysis to asses a business’s strengths, weaknesses, opportunities, and threats. This a critical task during the systems planning phase, as the SWOT analysis provides invaluable insight as to whether developing (or purchasing) a new system or application in feasible for business at the time. It also emphasizes the importance of planning in the IT industry in general, as major projects can cost businesses thousands if not millions of dollars in resources, and even more if the system is not planned properly!

Pure HTML5 & CSS3 Website

It was interesting to look back at my final project for this course, which was to create a website demonstrating as many of the HTML5 and CSS3 programming skills as I could. This course was virtually self-taught and relied heavily on my ability to acquire and apply concepts outlined in a textbook. One of the biggest challenges I had to overcome was learning how to position elements on a webpage so that they render properly regardless of screen size. I put a tremendous amount of work into this project because it incorporated one of my greatest passions – baking! At the time, I was proud of how it turned out. However, as you will see shortly, my website development skills became much more polished once I acquired a foundation in JavaScript!

JS & Bootstrap 4 Virtual Storefront

This website is my proudest programming achievement to date! It builds on my previous bakery website by implementing JavaScript and PHP to create a responsive, interactive, and dynamic user interface. By incorporating product modals, customers can view additional information about bakery products in pop-up windows on the same screen. Additionally, the entire site loads on a single page, meaning users never have to open a new window to view the home, about, or contact pages. Unlike my first bakery website, the use of PHP on this site allows the “Contact Us” form to forward data to the server! A sleek and simple design based on the principles of good UI and UX design principles outlined in my application development course, I love my new website!
