VB Mileage Calculator

This course gave me the opportunity to work with Visual Basic in the Visual Studio 2017 IDE. This assignment required the construction of a GUI application used to dynamically calculate the reimbursement of travel expenses incurred by a company’s employees. It tracks variables including distance traveled (km), regular reimbursement rate ($/km), flat reimbursement rate ($/km), number of days employees qualified for the flat reimbursement amount, and total number of days spent traveling.

JFrame Veterinary Record Updater

In preparation for the final exam in my Java course, I used Java, JFrame, and Eclipse to create this GUI application for a veterinary office. It allows users to sort through an array of veterinary records (objects of type custom class “Cat”) and input data for each. I am quite proud of this application as I constructed it completely on my own by integrating concepts from previous class assignments.

C Video Rental File System

The final project for my C programming course gave me the opportunity to use the skills I learned in class to develop a text-based database application for a video store. Using the application, users can input data for each movie available for rent, such as title, director, release date, etc. Each entry is appended to the end of a linked list, which in turn can be saved as a binary file. The application can then be used to load a previously saved binary file to work with all the movie entries saved within it. This was by far the most difficult program for me to create and required me to really exercise my research skills to complete it.

Figma Prototyping

This work sample is the mock-up of a mobile application, "GotSpot?" my group designed for our final application development & design project. This project was large in scale in that it required team collaboration to complete. I lead our team by assigning tasks to each member, performing regular check-ins with them, and updating our Trello board as tasks were completed. The result was a working digital prototype created using Figma. We incorporated the six principles of good user interface (UI) design to create a mobile application that allows motorists to rent parking spots from other app users.

Inventory Management System

This work sample demonstrates my ability to integrate various technologies including PHP, phpMyAdmin, and Laravel 8 to develop a web-based inventory management system. Whenever a category or item is created or edited, it is synced to a database. Finally, the user has the ability to upload and later edit item images to the server.

Blackjack Mobile Game

This work sample demonstrates my ability to utilize Java, Android Studio, and OOP principles to design a blackjack mobile game for Android devices. The design process behind this mobile application was two-fold: the game logic and the GUI. I had previously created a command-line blackjack game using C++ and OOP principles, so the logic was very much the same. However, the challenge this time was integrating that backend logic with a GUI (i.e. interactive buttons and images) users interact with to play the game. I found Android Studio rather intimidating when first learning it. However, after completing this project I feel a lot more confident using it to create mobile applications for Android devices.

JS & Bootstrap 4 Virtual Storefront

This website is one of my proudest programming accomplishments to date! It builds on my previous bakery website by implementing JavaScript and PHP to create a more responsive, interactive, and dynamic user interface. By incorporating product modals, customers can view additional information about bakery products in pop-up modules on the same screen. Additionally, the entire site loads on a single page, meaning users never have to open a new window to view the home, about, or contact pages. Unlike my first bakery website, the use of PHP on this site allows the “Contact Us” form to forward the completed form to a server. This sleek and simple design based on best practices for UX and UI make this site one of my most impressive GUI-based projects yet!
